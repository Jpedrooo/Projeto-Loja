# 🏺 L’Essence d’Or - E-commerce de Perfumes

## 📖 Sobre o Projeto
Este projeto consiste no desenvolvimento de um site de compras voltado para o mercado de perfumaria de luxo. O trabalho foi desenvolvido como parte da disciplina de **Programação Web**, sob orientação do professor **Adriano Firmo de Paiva**.

O objetivo principal foi aplicar conceitos fundamentais de **HTML5** e **CSS3** para criar uma interface funcional, navegável e esteticamente agradável.

---

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de todas as páginas.
* **CSS3:** Estilização visual, layout e design.

---

## 📄 Estrutura do Site
Conforme os requisitos solicitados, o site é composto pelas seguintes páginas:

* **Página Inicial (Home):**  Apresenta o nome do site no cabeçalho.
    * Exibe no mínimo dois produtos em destaque com imagem, descrição e link de compra.
    * Inclui uma seção de notícias do setor de perfumaria.
* **Listagem de Produtos:** Uma vitrine completa com o catálogo da loja.
* **Carrinho de Compras:** Simulação de uma tela de checkout contendo os produtos selecionados pelo usuário (apenas HTML).
* **Saiba Mais (Incluindo Fale Conosco):** * Página dedicada a informações institucionais.
    * Contém o **Formulário de Contato** com campos para preenchimento, atendendo ao requisito de "Fale Conosco".

---

## ⚙️ Funcionalidades e Requisitos Atendidos
* **Menu Navegável:** Todas as páginas possuem um menu que permite o tráfego entre os links locais (`.html`) de forma fluida.
* **Componentes Padronizados:** Uso de cabeçalho com imagem/logo e rodapé em todas as páginas.
* **Rodapé Informativo:** Contém os dados de contato do desenvolvedor (Nome, Telefone e E-mail).
* **Links de Ação:** Cada produto possui uma imagem ilustrativa, descrição e o link "Adicionar ao carrinho".
